<a name="dataReadyPush"></a>
### 数据准备完毕通知（票易通→客户)
```
POST 客户提供URL，系统配置后方可使用
```
* `文件加密密码为票易通分配`
* `文件地址为OSS地址，可直接访问拉取`

#### 参数

|字段名|字段|类型|是否必填|说明|
|---|---|---|---|---|
|**platformId**|**资金方编号**|**String(20)**|  <br>*必填*|资金方唯一编号，票易通分配|
|**timestamp**|**时间戳**|**String(20)**|  <br>*必填*|****|
|**taxNo**|**公司税号**|**String(50)**|  <br>*必填*|****|
|**fileUrl**|**文件地址**|**String(500)**|  <br>*必填*|****|
|**fileSignType**|**文件加密方式**|**String(50)**|  <br>*必填*|默认AES|
|**platformId**|**资金方编号**|**String(20)**|  <br>*必填*|****|


#### 响应参数

|字段|类型|说明|
|---|---|---|
|respCode|String|返回接口操作代码，0000表示成功，非0000表示错误代码。|
|respName|String|操作信息，调用成功返回“success”，失败返回错误代码对应的错误信息。|
|respMsg|String|返回操作中文描述信息|
|value|String|详细信息|

#### 响应示例

|字段|
|---|
|{"respCode":"0000","respName":"success","respMsg":"success","value":"success"}|

<a name="dataFinishNotify"></a>
### 文件格式
```
PUT  http://dvas-fat.daas-t.xforceplus.com/data/v1/finshNodify
```
* `文件后缀为.csv`
* `以换行符为单条数据切换`
* `每一行字段区分为","，例如123131414,112314,s,0009901824108,上海云励信息科技有限公司,19989898981,麦当劳,20191212,19889.00,111.00,20000.00,1,1,ext_info`

#### 字段说明

|字段名|描述|类型|说明|
|---|---|---|---|---|
|invoiceCode|发票代码(字符20位)|String|****|
|invoiceNo|发票号码(字符8位)|String|****|
|invoiceType|发票类型(字符2位)|String|s:增值税专用发票、ct:通行费增值税电子普通发票、v:机动车销售统一发票|
|purchaserTaxNo|购方税号(字符30位)|String|****|
|purchaserName|购方名称(字符200位)|String|*****|
|sellerTaxNo|销方税号(字符30位)|String|****|
|sellerName|销方名称(字符200位)|String|****|
|paperDrewDate|开票日期(字符8位)|String|****|
|amountWithoutTax|不含税金额(18,6)|String|****|
|taxAmount|税额(18,6)|String|****|
|amountWithTax|含税金额(18,6)|String|****|
|status|发票状态(字符2位)|String|0-作废 1-正常 2-红冲 3-失控异常|
|authStatus|认证状态(字符2位)|String|0-待认证 1-已勾选（签名确认）|
|ext|扩展字段(字符3000位)|String|****|
